# List available phone numbers

---

## 1. METADATA

| Property | Value |
| :--- | :--- |
| **HTTP Method** | GET |
| **Endpoint URL** | `https://services.leadconnectorhq.com/phone-system/numbers/location/:locationId/available` |
| **Scopes Required** | `phonenumbers.read` |
| **Authentication** | OAuth Access Token / Private Integration Token |
| **Token Type** | Sub-Account Token |

---

## 2. REQUEST

### Header Parameters

| Name | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| **Version** | `` | No |  |

### Path Parameters

| Name | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| **locationId** | `` | No |  |

### Query Parameters

| Name | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| **countryCode** | `` | No | numberTypes string Comma-separated list of phone number types to search for (e.g. local, tollFree, mobile) Example: local,tollFree |
| **numberTypes** | `` | No | firstPart string Filter numbers that begin with this digit pattern Example: 830 |
| **firstPart** | `` | No | lastPart string Filter numbers that end with this digit pattern Example: 825 |
| **lastPart** | `` | No | anywhere string Filter numbers that contain this digit pattern anywhere Example: 825 |
| **anywhere** | `` | No | smsEnabled boolean Filter for numbers with SMS capability Example: |
| **smsEnabled** | `` | No | mmsEnabled boolean Filter for numbers with MMS capability Example: |
| **mmsEnabled** | `` | No | voiceEnabled boolean Filter for numbers with voice capability Example: |
| **voiceEnabled** | `` | No |  |

N/A
