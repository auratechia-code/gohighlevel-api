# App Creation Guide

App Creation Guide
This guide walks you through creating a Marketplace App in HighLevel, from registering as a developer to setting up OAuth (scopes, redirect URLs, and client keys). It’s written to be practical and easy to follow, so you can build and test quickly, then publish when you’re ready.
What you’ll build
​
By the end of this document, you will have:
A Marketplace App created in HighLevel
A complete app listing profile (logo, description, screenshots, etc.)
OAuth configured (scopes + redirect/callback URL)
Client credentials generated (Client ID + Client Secret)
A clear understanding of where to manage versions, webhooks, and authentication settings
