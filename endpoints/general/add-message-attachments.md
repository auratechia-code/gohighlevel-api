# Add message attachments

Set attachments on an existing message (replaces existing). Maximum 5 URLs. Supported for Custom Call message type.

## Endpoint Information

> [!IMPORTANT]
> **Method:** `PUT`  
> **URL:** ``

## Code Examples

